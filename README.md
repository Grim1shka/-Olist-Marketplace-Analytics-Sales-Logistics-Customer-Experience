Olist Marketplace Analytics: Sales, Logistics and Customer Experience
Аналитика маркетплейса Olist: продажи, логистика и клиентский опыт

------------------------------------------------------------

1. Описание проекта

Этот проект представляет собой аналитическое решение на основе данных маркетплейса Olist.

Цель — провести полный цикл анализа бизнеса:
- изучить продажи и выручку
- проанализировать поведение клиентов
- оценить работу продавцов
- исследовать географию заказов
- изучить логистику и время доставки
- проверить влияние доставки на оценки клиентов

Используемые инструменты:
- PostgreSQL для обработки данных
- Power BI для визуализации
- DAX для расчета метрик

------------------------------------------------------------

2. Бизнес-задача

Основные вопросы бизнеса:
- какие факторы влияют на выручку
- какие категории товаров приносят больше дохода
- как распределены продажи по регионам
- насколько эффективно работает логистика
- влияет ли скорость доставки на удовлетворенность клиентов

------------------------------------------------------------

3. Источники данных

Используется набор данных Olist, включающий:
- заказы
- товары
- продавцы
- клиенты
- платежи
- отзывы
- географию
- категории товаров

------------------------------------------------------------

4. Подготовка данных

Данные разделены на слои:

RAW слой:
Сырые таблицы без изменений (orders, products, customers и др.)

ANALYTICS слой:
Подготовленные витрины:
- fact_sales
- orders_clean
- dim_products
- delivery_analysis_v2
- review_analysis

Основные преобразования:
- очистка дат
- объединение таблиц через JOIN
- расчет выручки (price + freight_value)
- расчет времени доставки
- удаление аномалий доставки

------------------------------------------------------------

5. Логика расчетов

Выручка:
price + freight_value

Время доставки:
delivery_days = дата доставки - дата покупки

Аномалии:
- отрицательные значения доставки
- доставки более 60 дней

------------------------------------------------------------

6. Модель данных в Power BI

Измерения:

Дополнительные таблицы:

8. Основные выводы

- продажи имеют выраженную сезонность
- основная выручка сосредоточена в нескольких регионах
- небольшое число продавцов формирует значительную часть выручки
- среднее время доставки около 11–12 дней
- скорость доставки влияет на оценки клиентов
